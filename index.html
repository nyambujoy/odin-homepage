A Tic-Tac-Toe Game

This project is a simple implementation of the classic Tic-Tac-Toe game, created using JavaScript, HTML, and CSS. The game allows two players to compete by taking turns marking spaces in a 3x3 grid. The first player to align three of their marks either horizontally, vertically, or diagonally wins the game.
